This Bash script is a lightweight system monitoring tool for Linux that provides real-time insights into system performance. It continuously tracks and displays essential system metrics, including CPU usage per core, memory consumption, network speed, battery percentage, and system load averages. 


main()
{
    
    get_cpu_usage
    get_network_base

    while true; do
        sleep 1
        clear
        get_network_usage
        get_cpu_usage
        get_power_info
        get_loadavg_info
        get_mem_info
    done

}

The script refreshes every second, making it ideal for users who need a quick and clear overview of their system's status directly in the terminal. The script collects data from /proc and /sys files, ensuring accurate and up-to-date monitoring. Network speed is visualized using a simple bar chart, allowing users to see fluctuations over time.
